== Description ==

This plugin shows a user table in /wp-admin/ under the Users menu item. It has functions to filter users by user role and order by username and name.

== Installation ==

1. Upload the `wppro-user-filter` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Take a look at the new `User Filter` menu under the `User` menu in `wp-admin`
